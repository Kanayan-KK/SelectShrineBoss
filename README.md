# Select Shrine Boss

Elin Mod

[SteamWorkshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3618911878)

## Description

【概要】
争いの祠を使用時に召喚されるボスモンスターをリストから選択可能にします。

【修正内容】
争いの祠使用時に召喚可能モンスター名のリストを表示します。
リスト上の選択したモンスターをボスモンスターとして生成します。

【注意】
リストの内容はバイオームやネフィア危険度の影響を受けます。

ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー

[Overview]
Allows you to select the boss monster summoned when using the Shrine of Strife from a list.

[Features]
Displays a list of summonable monster names when the Shrine of Strife is used.
Spawns the monster selected from the list as the boss.

[Note]
The contents of the list are affected by the current Biome and Nefia Danger Level.
